# Exercice 01: Hello Print - Défis

Maintenant que vous avez vu les exemples, essayez ces défis !

## Défi 1: Présentation personnelle

Créez un programme qui affiche :
- Votre nom
- Votre âge
- Votre ville
- Pourquoi vous apprenez Python

Format attendu :
---
Nom : [votre nom]
Âge : [votre âge]
Ville : [votre ville]
Objectif : [pourquoi vous apprenez Python]
---

## Défi 2: Banner ASCII Art

Créez un banner avec votre pseudo ou votre nom en ASCII art.

Exemple :
  _____ __   __ _______ _     _  _____  __   _
 |_____]   \_/      |   |_____| |     | | \  |
 |          |       |   |     | |_____| |  \_|

Vous pouvez utiliser un générateur en ligne : https://patorjk.com/software/taag/

## Défi 4: Affichage progressif

Créez un affichage qui simule un chargement :

[*] Initializing...
[*] Loading modules...
[*] Connecting to target...
[*] Scanning ports...
[+] Scan complete!

Utilisez les symboles :
[*] = Information
[+] = Succès
[-] = Échec
[!] = Avertissement

## Conseils

1. Expérimentez avec différents caractères Unicode
2. Utilisez print() avec end="" pour continuer sur la même ligne
3. Utilisez sep pour changer le séparateur
4. Utilisez \n pour les nouvelles lignes
5. Utilisez \t pour les tabulations
6. Les caractères * peuvent être répétés : "*" * 50
